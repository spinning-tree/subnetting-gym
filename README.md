# SUBNETTING GYM

Aplicación para practicar cálculos de direccionamiento IPv4 y subnetting.

## CARACTERÍSTICAS

- Generación dinámica de ejercicios de subnetting.
- Cálculo de rangos de hosts, direcciones de broadcast y subredes específicas.
- Interfaz basada en Material Design 3 con modo oscuro permanente.
- Desarrollo en Vanilla JavaScript sin dependencias externas.
- Diseño responsivo adaptado a dispositivos móviles.

## EJERCICIOS INCLUIDOS

- Identificación de rangos de hosts válidos según prefijo CIDR.
- Localización de direcciones de broadcast con máscaras decimales.
- Determinación de la última dirección de host utilizable.
- Identificación de la subred de origen para un host dado.
- Cálculo de la primera dirección de host para la enésima subred de un bloque.
- Conteo de subredes y hosts disponibles según la máscara aplicada.

## INSTALACIÓN Y USO

La aplicación consta de un único archivo HTML autocontenido. No requiere compilación ni servidores especializados.

1. Descargue el archivo index.html.
2. Abra el archivo en cualquier navegador web moderno.

## REQUISITOS DEL SISTEMA

- Navegador compatible con HTML5, CSS3 y ES6 (Chrome, Firefox, Safari, Edge).
- No requiere conexión a internet tras la carga inicial.
